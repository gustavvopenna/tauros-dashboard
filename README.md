# Dashboard - Tauros
Prueba técnica para el puesto de Frontend Developer en Tauros.io

## Sitio público

El deploy se realizó con Vercel y lo pueden encontrar en:
[este link.](https://tauros-dashboard.vercel.app/)

## Instalación

#### Clonar el proyecto
```
git clone 
```
    
#### Instalar las dependencias
```
yarn add
```
#### Correr el proyecto
```
yarn start
```
